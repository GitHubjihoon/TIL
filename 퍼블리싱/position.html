놀랍게도 position개념을 잊어버렸다!

CSS의 position 속성은 엘리먼트가 브라우저 화면에 어디에 배치되는가를 결정합니다. 기본 값 : static 변경 값 : relative _ 관계다른 position과의 관계를 나타내는 속성??? (추후 정보 업데이트 필요 2021.12.07) 변경 값 : absolute _ fixed :

absolute

부모 엘리먼트로 부터 자유로운 독립된 배치문맥(positioning context) 위에 쌓인 레이어 개념 브라우저 화면(viewport) 상 어디든 자유롭게 배치 가능 단, 상위 엘리먼트중 position속성이 relative가 있을경우 해당 요소를 대지로 삼아 offset속성(top, left, bottom, right)이 적용 됨.

위치 잡기

absolute를 사용해서 사진 위에 텍스트를 부유시키는것에 성공했다 하지만 문제에 도달핬습니다. 그 문제는 바로 top50%; left50% 를 적용하니 중앙에 배열되는것이 아닌 글씨의 좌상단 기준으로 시작이 되어 삐뚤어지는 현상이 발생했습니다. 2021.12.08
